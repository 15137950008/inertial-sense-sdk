64 /*
65 Packet Overview
66 
67 Byte
68 0 Packet start byte
69 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
70 2 Packet counter (for ACK and retry)
71 3 Packet flags
72 
73 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
74 4-7 Data identifier
75 8-11 Data length
76 12-15 Data offset
77 16-19 Data start
78 (n-8)-(n-5) Last piece of data
79 // end data
80 
81 n-4 Reserved
82 n-3 Checksum high byte
83 n-2 Checksum low byte
84 n-1 Packet end byte
85 */
